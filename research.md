# Research

## Crop Type Mapping Using Sentinel-1 and Sentinel-2 in Mali

### Overview

This research investigates crop type mapping in heterogeneous agricultural systems of Dioila, Mali using Sentinel-1 SAR and Sentinel-2 optical imagery.

### Research Questions

* How effectively can crop types be distinguished using Sentinel-1 and Sentinel-2 individually and in combination?
* How does classification accuracy vary spatially and temporally?
* Which spectral, temporal, and textural features contribute most to crop discrimination?

### Methods

* Random Forest Classification
* Google Earth Engine
* ArcGIS Pro
* Python
* Sentinel-1 SAR
* Sentinel-2 Optical Data
